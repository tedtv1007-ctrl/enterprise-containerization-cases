# Rocky Linux 10 安裝 Docker CE 指南

本專案提供在 Rocky Linux 10 上安裝 Docker CE (Community Edition) 的詳細步驟。

## 系統需求
- Rocky Linux 10 (Red Quartz)
- 具有 sudo 權限的使用者

## 安裝步驟

### 1. 移除舊版本 (選用)
如果系統中已存在舊版的 Docker，請先將其移除：

```bash
sudo dnf remove docker \
                  docker-client \
                  docker-client-latest \
                  docker-common \
                  docker-latest \
                  docker-latest-logrotate \
                  docker-logrotate \
                  docker-engine
```

### 2. 設定存儲庫 (Repository)
安裝 `dnf-plugins-core` 並新增 Docker 的官方存儲庫：

```bash
sudo dnf -y install dnf-plugins-core
sudo dnf config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
```

### 3. 安裝 Docker Engine
安裝最新版本的 Docker CE、CLI 與相關插件：

```bash
sudo dnf install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

### 4. 啟動 Docker
設定 Docker 隨系統啟動並立即執行：

```bash
sudo systemctl enable --now docker
```

### 5. 驗證安裝
執行測試容器以確認安裝成功：

```bash
sudo docker run hello-world
```

## 進階設定

### 免 sudo 執行 Docker
若要讓目前使用者不需 `sudo` 即可執行 Docker 指令：

```bash
sudo usermod -aG docker $USER
newgrp docker
```
