# Rocky Linux 10 安裝 Kubernetes (Kubeadm) 指南

本指南接續 Docker 安裝教學，說明如何在 Rocky Linux 10 上使用 `kubeadm` 佈署 Kubernetes 叢集。

## 系統準備 (所有節點)

### 1. 停用 Swap (K8S 強制要求)
```bash
sudo swapoff -a
# 永久停用：編輯 /etc/fstab，註解掉 swap 相關行
sudo sed -i '/ swap / s/^\(.*\)$/#\1/g' /etc/fstab
```

### 2. 設定 SELinux 為 Permissive 模式
```bash
sudo setenforce 0
sudo sed -i 's/^SELINUX=enforcing$/SELINUX=permissive/' /etc/selinux/config
```

### 3. 開放核心模組與網路過濾
```bash
cat <<EOF | sudo tee /etc/modules-load.d/k8s.conf
overlay
br_netfilter
EOF

sudo modprobe overlay
sudo modprobe br_netfilter

# 設定所需的 sysctl 參數
cat <<EOF | sudo tee /etc/sysctl.d/k8s.conf
net.bridge.bridge-nf-call-iptables  = 1
net.bridge.bridge-nf-call-ip6tables = 1
net.ipv4.ip_forward                 = 1
EOF

# 立即套用參數
sudo sysctl --system
```

## 安裝 Kubernetes 套件 (所有節點)

### 1. 新增 Kubernetes Repo (pkgs.k8s.io)
這裡以 v1.31 為例（請根據需求調整版本號）：

```bash
cat <<EOF | sudo tee /etc/yum.repos.d/kubernetes.repo
[kubernetes]
name=Kubernetes
baseurl=https://pkgs.k8s.io/core:/stable:/v1.31/rpm/
enabled=1
gpgcheck=1
gpgkey=https://pkgs.k8s.io/core:/stable:/v1.31/rpm/repodata/repomd.xml.key
exclude=kubelet kubeadm kubectl cri-tools kubernetes-cni
EOF
```

### 2. 安裝套件
```bash
sudo dnf install -y kubelet kubeadm kubectl --disableexcludes=kubernetes
sudo systemctl enable --now kubelet
```

## 設定 Container Runtime (以 Containerd 為例)

Kubernetes 需要透過 CRI 溝通。由於你之前安裝了 Docker，它已經附帶了 `containerd`。

### 1. 產生預設設定並啟用 Cgroup Driver
```bash
sudo mkdir -p /etc/containerd
containerd config default | sudo tee /etc/containerd/config.toml
# 將 SystemdCgroup 設定為 true
sudo sed -i 's/SystemdCgroup = false/SystemdCgroup = true/g' /etc/containerd/config.toml
sudo systemctl restart containerd
```

## 初始化叢集 (僅在 Master 節點)

執行初始化（請替換 `<MASTER_IP>` 為你的主機 IP）：
```bash
sudo kubeadm init --pod-network-cidr=10.244.0.0/16 --apiserver-advertise-address=<MASTER_IP>
```

初始化成功後，依照畫面指示設定 `kubectl` 權限：
```bash
mkdir -p $HOME/.kube
sudo cp -i /etc/kubernetes/admin.conf $HOME/.kube/config
sudo chown $(id -u):$(id -g) $HOME/.kube/config
```

## 安裝網路插件 (CNI)
推薦安裝 Flannel 或 Calico，這裡以 Flannel 為例：
```bash
kubectl apply -f https://github.com/flannel-io/flannel/releases/latest/download/kube-flannel.yml
```
